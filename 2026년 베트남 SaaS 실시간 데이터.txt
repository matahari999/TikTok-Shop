수수료 폭탄이 지금 터짐 (핵심)

- 2025년: 수수료 2~3%
- 2026년 3월: 마켓플레이스 셀러 12.5%로 폭등
- 트랜잭션 수수료 별도 6%
- 주문당 VND 3,000 추가

→ 셀러들이 "내가 지금 손해 보고 파는 건지도 모른다" 상태

---
진짜 공백

기존 툴들은 주문관리에 집중. 하지만 지금 셀러들이 진짜 필요한 건:

▎ "수수료 12.5% + 트랜잭션 6% + VND 3,000 + 배송비 + 원가 다 빼면 실제로 얼마 남는지"

이걸 자동으로 계산해주는 베트남어 + 저렴한 + 모바일 친화적 툴이 없음.

---
만들면 팔리나?

예, 조건이 맞습니다:
- 수수료 변경이 지금 막 터진 통증 → 타이밍 완벽
- 베트남 TikTok Shop 셀러 수백만 명
- 기존 툴은 이 기능에 집중 안 함
- 가격 100,000~200,000 VND/월 ($4~8) 이면 진입장벽 낮음

제품 방향 (MVP)

"베트남 TikTok Shop 수익 계산기 + 대시보드"
1. TikTok Shop API 연동 → 주문 자동 수집
2. 수수료 자동 계산 (카테고리별 12.5% 등)
3. SKU별 실제 마진 표시
4. 마진 0% 이하 상품 경보
